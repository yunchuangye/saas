## 一、项目概述

GuJia.App 是一个**房产评估 SaaS 平台**，连接个人客户、银行/投资机构、评估公司三方，通过竞价机制撮合评估需求与服务供给，并提供 AI 辅助估价、数据爬取、报告管理等功能。

**技术栈**

| 层级 | 技术 |
|------|------|
| 前端 | Next.js 16 + TypeScript + TailwindCSS + shadcn/ui |
| 后端 | Express + tRPC + Drizzle ORM |
| 数据库 | MySQL 8.0（含 217 万条房屋数据）+ Redis |
| AI | OpenAI API（GPT-4o）|
| 文件存储 | 本地 / 阿里云 OSS（可配置）|

**用户角色**

| 角色 | 说明 |
|------|------|
| `customer` | 个人客户，提交评估申请、查看报告 |
| `appraiser` | 评估公司，竞价承接项目、出具报告 |
| `bank` | 银行机构，发起评估需求、审核报告 |
| `investor` | 投资机构，发起评估需求、审核报告 |
| `admin` | 运营管理，管理用户/数据/系统配置 |
